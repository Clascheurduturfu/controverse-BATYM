# BATYM | La Manhattanisation de Paris

Ce projet est une étude de controverse sur la construction de gratte-ciels à Paris.

## Technologies utilisées
- **Vite** : Pour un développement rapide et performant.
- **React** : Pour une interface dynamique et réactive.
- **Tailwind CSS** : Pour un design moderne et responsive.
- **Framer Motion** : Pour des animations fluides et immersives.
- **Lucide React** : Pour une iconographie claire.

## Installation et Lancement

1. Installez les dépendances :
   ```bash
   npm install
   ```

2. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

3. Pour construire le site statique :
   ```bash
   npm run build
   ```

## Structure
- `src/App.tsx` : Contient l'intégralité de la structure et du contenu de l'analyse.
- `src/index.css` : Styles globaux et configurations visuelles.
- `Controverse.pdf` : Document source de l'analyse.
